# Multilingual AI Career Marketer Agent

A production-ready, knowledge-grounded cloud AI Agent built using Microsoft Azure Foundry. This system is engineered to solve recruitment screening friction by serving as an articulate, multilingual technical representative.

## 🚀 Core Capabilities
- **Multilingual Code-Switching**: Automatically detects and transitions communication fluently across official South African languages (including Sepedi and English) with context-aware technical terminology.
- **Retrieval-Augmented Generation (RAG)**: Zero-hallucination execution. Grounded via an Azure AI Search vector index parsing unstructured academic transcripts and enterprise internship summaries.
- **Enterprise Engineering Persona**: Programmed to structurally convey academic excellence (85% Systems Development 3 / 86% Android App Development) and specialized cloud skills.

## 🛠️ Architecture & Tech Stack
- **AI Orchestration Framework**: Microsoft Azure AI Foundry / gpt-5 foundational engine
- **Knowledge Vectorization**: Azure AI Search index (`index_shadi_career_knowledge`)
- **Speech Pipeline Integration**: Azure Neural Text-to-Speech & Speech-to-Text streaming
- **Underlying Capabilities**: C# / Microsoft Foundry SDK / AI-900 Core Architecture

## 📋 System Configuration (YAML Specifications)
The complete orchestration rules, metadata, and identity schemas are preserved in the `agent-config.yaml` file within this repository.
